## Why I Built This

I was studying Motion's product — an AI command center for creative strategists — and I kept thinking: *this is exactly the problem I want to solve.*

So instead of just applying, I spent 3 days building a mini version of what Motion does at its core.

**CreativeIQ** takes your ad creative performance data and tells you:
- 🏆 Which creatives are winning and exactly why
- ⚠️ Which creatives are fatiguing and need replacing
- 🔍 What patterns separate winners from losers
- 🚀 Exactly what to do next
- 🇮🇳 How these patterns apply to India's D2C market

## How It Works
```
Upload CSV → AI Agent Analyzes → Creative Intelligence Report
```

1. Upload your ad creative data as CSV
2. CreativeIQ sends it to a Gemini 2.5 AI agent
3. Get a structured intelligence report in seconds — winners, fatigue alerts, patterns, recommendations

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite |
| AI Agent | Google Gemini 2.5 Flash |

## Sample Data Format

Your CSV should have these columns:
```csv
creative_name,spend,impressions,clicks,ctr,roas,format,platform
Founder_Story_01,5000,120000,6000,5.0,6.2,video,TikTok
Question_Hook_02,3200,95000,4275,4.5,5.1,video,Meta
```
